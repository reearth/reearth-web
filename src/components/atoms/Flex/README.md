Flex. This component is a global component.

## Usage

```jsx
import Flex from "@reearth/components/atoms/Flex";

<Flex wrap="wrap" direction="colum" justify="space-between" align="center">
	...
</Flex>
```

- justify = justifyContent

- wrap = flexWrap

- direction = flexDirection
- basis = flexBasis
- grow = flexGrow
- shrink = flexShrink
- flex = flex
- gap = gap
- className
- onClick: function
- children: ReactNode
